Welcome to CrazyTicket Bot's documentation!
===========================================

Summary
=======

CrazyTicket Bot is an app that allows broadcasters to sell tickets for a Hidden Cam show on Chaturbate. Commands for broadcasters and moderators are available. The app includes options for a tip goal and countdown goal timer.
